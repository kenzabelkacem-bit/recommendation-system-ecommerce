#  Système de recommandation intelligent pour une plateforme e-commerce

## Présentation

Ce projet a été réalisé dans le cadre d'un module de **Gestion de Projet** et de **Génie Logiciel**.

L'objectif est de concevoir un **système de recommandation intelligent** destiné à une plateforme e-commerce spécialisée dans la vente de produits électroniques. Le système permet de proposer automatiquement des recommandations personnalisées en fonction des préférences et du comportement des utilisateurs afin d'améliorer leur expérience d'achat.

Le projet couvre l'ensemble des phases de conception d'un projet logiciel, depuis l'analyse des besoins jusqu'à la planification du développement.

---

# Objectifs du projet

- Concevoir un système de recommandation intelligent.
- Analyser les besoins fonctionnels et non fonctionnels.
- Modéliser l'architecture du système avec UML.
- Planifier le projet à l'aide de Microsoft Project.
- Définir les ressources, les coûts et les délais.
- Identifier le chemin critique.
- Produire une documentation complète conforme aux bonnes pratiques de gestion de projet.

---

# Fonctionnalités prévues

Le système permet notamment de :

- créer un compte utilisateur ;
- se connecter à la plateforme ;
- consulter le catalogue des produits ;
- rechercher des produits ;
- recevoir des recommandations personnalisées ;
- consulter les détails d'un produit ;
- ajouter des produits au panier ;
- gérer son profil utilisateur ;
- enregistrer l'historique des recommandations.

# Intelligence artificielle

Le projet est basé sur le principe des **systèmes de recommandation**.

Les recommandations sont générées à partir des préférences des utilisateurs et de leur historique afin de proposer des produits susceptibles de les intéresser.

Le système pourra être enrichi par la suite avec différents algorithmes de Machine Learning tels que :

- Filtrage collaboratif
- Filtrage basé sur le contenu
- Similarité cosinus
- K-Nearest Neighbors (KNN)
- Deep Learning
